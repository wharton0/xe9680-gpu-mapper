# 🎉 Dell XE9680 GPU 映射查询程序 - Rust 桌面版

## ✨ 使用说明（超级简单！）

### 第一次使用：

1. **双击 `run.bat`** → 自动编译并启动程序
   
   或者直接运行：
   ```bash
   cargo build --release
   ```

### 之后每次使用：

**直接双击这个文件即可：**
```
target\release\xe9680-gpu-mapper.exe
```

就像打开 HTML 文件一样简单！会打开一个窗口显示界面。

---

## 🔍 如何使用

1. **打开程序** → 会直接显示一个窗口（无黑色命令行框），包含：
   - 🔍 搜索框
   - 🖼️ GPU 物理位置示意图（左侧）
   - 📋 查询结果详情（右侧）

2. **输入查询**：在搜索框输入以下任意内容：
   - `0` → 查找 Minor Number 为 0 的 GPU
   - `3B` → 查找 Bus ID 为 3B 的 GPU  
   - `gpu1` → 查找 Module ID 为 gpu1 的 GPU
   - `s28` → 查找 Slot 为 s28 的 GPU
   - `psb1` → 查找 PSB 为 psb1 的所有 GPU

3. **查看结果**：
   - 🔴 左侧示意图中匹配的 GPU 会**红色高亮**并放大
   - 📝 右侧显示详细信息表格

## ⚡ 快速测试

运行程序后，试试输入这些：

- ✅ `0` → 应高亮 GPU 2 (Bus ID: 19)
- ✅ `gpu1` → 应高亮 GPU 1 (Bus ID: 4C)
- ✅ `3B` → 应高亮 GPU 4
- ✅ `psb1` → 应高亮 2 个 GPU
